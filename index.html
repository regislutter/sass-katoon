Icons

The Sass Framework is using Iconic to generate icons. Iconic can render SVG in image tag, import SVG file with JavaScript or even be used as Webfont.

Read the documentation in Iconic guides

See the complete list of icons

Simple icons SVG

Display icons easily by putting the SVG or PNG file in a image tag.

    <img src="iconic/svg/smart/bar-chart.svg">
Power-up icon

Import the SVG file with the Iconic Javascript library

Needs iconic.min.js to import SVG

    <img class="iconic" style="fill:blue;stroke:red;"
        data-src="iconic/svg/smart/envelope.svg">
Responsive icons

Needs iconic.min.js to import SVG

    <img class="iconic iconic-sm"
        data-src="iconic/svg/smart/sun.svg">
    <img class="iconic iconic-md"
        data-src="iconic/svg/smart/sun.svg">
    <img class="iconic iconic-lg"
        data-src="iconic/svg/smart/sun.svg">
Style and animate with CSS

Needs iconic.min.js to import SVG

   <img class="iconic iconic-custom"
        data-src="iconic/svg/smart/bar-chart.svg">

    .iconic-custom {
        .iconic-bar-chart-bar-1  {
            fill: $green;
            stroke: $green;
        }

        .iconic-bar-chart-bar-2  {
            fill: $blue;
            stroke: $blue;
        }

        .iconic-bar-chart-bar-3  {
            fill: $purple;
            stroke: $purple;
        }

        .iconic-bar-chart-bar-4  {
            fill: $red;
            stroke: $red;
        }

        .iconic-bar-chart-bar-5  {
            fill: $orange;
            stroke: $orange;
        }

        .iconic-bar-chart-axes {
            stroke: $gray;
        }
    }
Icons with font

You can use Iconic as a font. This gives access to functionalities like adding text next to the icon (ideal for buttons), or having text as a fallback to the icon.

Needs iconic.min.js to import SVG
Needs to import "iconic/iconic-glyphs" in app.scss


Like
Like (fallback) 
    <span class="iconic"
        data-glyph="heart"
        title="heart"
        aria-hidden="true"></span>

    <span class="iconic"
        data-glyph="thumb-up"
        title="like"
        aria-hidden="true"> Like</span>

    <span class="iconic iconic-text-replace"
        data-glyph="thumb-up"
        title="like"
        aria-hidden="true"> Like (fallback)</span>
Attributes

Some icons accept attributes as parameters to add information in the icon itself, like a file format.

Needs iconic.min.js to import SVG

    <img class="iconic"
        data-src="iconic/svg/smart/file.svg"
        data-file-extension="CCP4">
